Welcome to FLAME AI

Empowering first responders and residents with comprehensive fire detection technology, ensuring swift and efficient response to protect lives and property   

Machine Learning

Using a hybrid approach that utilizes a YOLOv5 computer vision model to detect people and a logistic regression model trained on fire sensory data, we predict the probability of a fire and the number of people in each area.

YOLOv5

We used a pre-trained YOLOv5 image classification model that we fine-tuned on fire images. This process yielded a refined YOLOv5 variant tailored for fire detection.

CV2

We used the OpenCV (cv2) library to capture and preprocess video stream data, converting each frame into a format suitable for analysis. This processed data was then fed into a machine learning model designed for the real-time classification of images to accurately identify the presence of fires and people within the video frames.
